# Уеб базирана платформа за анализ на техника при гребане

## Описание

Система за анализ на техника при гребане чрез компютърно зрение и обработка на видео данни.

## Функционалности

- **Анализ на видео** – извличане на ключови точки от MediaPipe
- **Референтен модел** – обучение от видеа с правилна техника
- **Чатбот** – съвети за техника, планове и спортна подготовка

## Стартиране

```powershell
cd backend
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
python run.py
```

Отворете http://localhost:8000

## Чатбот (опционално)

За пълна функционалност на чатбота:

1. Копирайте `backend/.env.example` в `backend/.env`
2. Добавете OpenAI API ключ: `OPENAI_API_KEY=sk-...`
3. Ключ от platform.openai.com

Без ключ чатботът дава основни съвети на базата на предварително зададени отговори.
