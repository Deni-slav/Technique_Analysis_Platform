# Уеб базирана платформа за анализ на техника при кану-каяк

## Описание

Система за анализ на техника при кану-каяк чрез компютърно зрение и обработка на видео данни. Платформата извлича биомеханични параметри от тренировъчни видеа и предоставя обективна оценка на техниката.

## Функционалности

- **Анализ на видео** – извличане на ключови точки от тялото чрез MediaPipe
- **Биомеханични метрики** – ротация на торса, честота (SPM), drive/recovery съотношение, симетрия
- **Референтен модел** – обучение от видеа с правилна техника и сравнение с тях
- **Чатбот** – съветник за техника, тренировъчни планове и спортна подготовка

## Технологии

- **Backend:** Python, FastAPI, MediaPipe, OpenCV, NumPy/SciPy
- **Frontend:** HTML, CSS, JavaScript (без framework)
- **AI:** OpenAI GPT (чатбот), MediaPipe Pose Landmarker (поза)

## Стартиране

```powershell
cd backend
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
python run.py
```

Отворете http://localhost:8000

## Чатбот (опционално)

За пълна функционалност на чатбота:

1. Копирайте `backend/.env.example` в `backend/.env`
2. Добавете OpenAI API ключ: `OPENAI_API_KEY=sk-...`
3. Ключ от platform.openai.com

Без ключ чатботът дава основни съвети на база предварително зададени отговори.

## Структура

```
├── backend/
│   ├── analysis/        # Биомеханичен анализ и референтен модел
│   ├── app/routers/     # FastAPI endpoints
│   └── chat/            # Чатбот промпти
├── frontend/            # HTML/CSS/JS интерфейс
├── reference_videos/    # Референтни видеа за обучение на модела
├── uploads/             # Качени видеа за анализ
└── outputs/             # Резултати от анализи (JSON)
```
